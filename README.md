
# Pokedex

Simple Pokedex Built with PokeAPI.

## Demo

https://nikshaan.github.io/Pokedex/


## Tech Stack

* React.js
* Tailwind CSS
* Framer Motion
* Chart.js


## Features

* Search Pokemon
* Filter by type
* Filter by generation
* Chart of pokemon's stats
* Scroll to top
* Lazy Loading Images
* API calling
